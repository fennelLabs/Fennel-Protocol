Expand description

Drain panicking on error

Logger requires a root drain to handle all errors (Drain::Error == ()), Fuse will wrap a Drain and panic if it returns any errors.

Note: Drain::Err must implement Display (for displaying on panic). It’s easy to create your own Fuse drain if this requirement can’t be fulfilled.
